# Análise de Requisições Diárias

## Contexto
Análise do volume diário de requisições com foco em identificar padrões,
tendências e sazonalidade para apoiar decisões operacionais.

## Objetivo
Explorar dados temporais utilizando Python e gerar insights relevantes
para o negócio.

## Tecnologias
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Principais Análises
- Evolução temporal das requisições
- Média móvel
- Análise por dia da semana
- Identificação de padrões sazonais

## Próximos Passos
- Detecção de anomalias
- Previsão de demanda
- Criação de dashboards
